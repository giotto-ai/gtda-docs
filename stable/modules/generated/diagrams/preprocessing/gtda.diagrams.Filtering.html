Filtering¶

class gtda.diagrams.Filtering(homology_dimensions=None, epsilon=0.01)[source]¶

Filtering of persistence diagrams.

Filtering a diagram means removing all points whose distance from the diagonal is less than or equal to a certain cutoff value which can be interpreted as (\(1/\sqrt{2}\) times) the “minimum amount of persistence” required from points in the filtered diagram.

Parameters

  • homology_dimensions (iterable or None, optional, default: None) – When set to None, subdiagrams corresponding to all homology dimensions seen in fit will be filtered. Otherwise, it contains the homology dimensions at which filtering should occur.

  • epsilon (float, optional, default: 0.01) – The cutoff value controlling the amount of filtering.

homology_dimensions_¶

If homology_dimensions is set to None, then this is the list of homology dimensions seen in fit, sorted in ascending order. Otherwise, it is a similarly sorted version of homology_dimensions.

Type

list

See also

Scaling, Amplitude, PairwiseDistance, gtda.homology.VietorisRipsPersistence
